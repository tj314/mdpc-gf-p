\doxysubsection{Detailed Description}
\mbox{\hyperlink{structDecodeBounds}{Decode\+Bounds}} struct holds values used in decoding. 

These values are needed in each iteration of decoding. They can be calculated from the parameter q and do not change until this parameter changes. The values are calculated using the \mbox{\hyperlink{structDecodeBounds_aa8149cc21797147a031f238ffcd615fb}{set()}} method. The values are uninitialized until it is called.
