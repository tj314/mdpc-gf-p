Decrypt a ciphertext. 

Decryption is achieved as follows\+:
\begin{DoxyEnumerate}
\item Ciphertext is decoded using the generated \mbox{\hyperlink{classCode}{Code}} instance to obtain the error vector used in encryption.
\item The error vector is subtracted from the ciphertext.
\item Plaintext is read as the first k positions from ciphertext. Decryption will fail if\+:
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Ciphertext is not of length 2$\ast$k.
\item Decoding fails.
\end{DoxyItemize}

Encrypt a plaintext. 

Encryption is achieved as follows\+:
\begin{DoxyEnumerate}
\item Plaintext is encoded using the generated \mbox{\hyperlink{classCode}{Code}} instance.
\item An error vector is generated.
\item Encoded plaintext is summed with the error vector. Encryption will fail if and only if the plaintext length is not k. Ciphertext is of length 2$\ast$k.
\end{DoxyEnumerate}
